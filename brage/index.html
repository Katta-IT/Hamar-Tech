HTML
Enkelt Middels Vanskelig

HTML står for HyperText Markup Language, og det er et standardisert markeringsspråk som brukes til å utvikle og presentere innholdet på nettsider. HTML brukes til å strukturere og organisere informasjon ved hjelp av ulike elementer og koder

CSS
Enkelt Middels Vanskelig

CSS står for Cascading Style Sheets, og det er et stilarksspråk som brukes til å definere utseendet og formgivningen av HTML-dokumenter. Mens HTML tar seg av strukturen og innholdet på en nettside, er CSS ansvarlig for presentasjonen og stiliseringen av dette innholdet.

JS
Enkelt Middels Vanskelig

JS står for JavaScript, og det er et høynivå programmeringsspråk som hovedsakelig brukes i webutvikling for å gjøre nettsider interaktive og dynamiske. JavaScript kan kjøres direkte i nettleseren og utføres på klientens datamaskin.
