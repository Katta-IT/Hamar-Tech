Hva er CSS?

CSS står for Cascading Style Sheets
CSS beskriver hvordan HTML-elementer skal vises på skjerm, papir eller i andre medier
CSS sparer mye arbeid. Den kan kontrollere utformingen av flere nettsider samtidig Eksterne stilark lagres i CSS-filer

Hvorfor bruke CSS?

CSS brukes til å definere stiler for nettsidene dine, inkludert design, layout og variasjoner i visning for ulike enheter og skjermstørrelser.

CSS Eksempel

CSS løste et stort problem

HTML var ALDRI ment å inneholde koder for formatering av en nettside!
HTML ble laget for å beskrive innholdet på en nettside, som:
<h1>Dette er en overskrift </h1>
<p>Dette er et avsnitt.</p>
Da tagger som <font> og fargeattributter ble lagt til HTML 3.2-spesifikasjonen, startet det et mareritt for webutviklere. Utvikling av store nettsider, hvor fonter og fargeinformasjon ble lagt til hver eneste side, ble en lang og kostbar prosess.
For å løse dette problemet opprettet World Wide Web Consortium (W3C) CSS.
CSS fjernet stilformateringen fra HTML-siden!

CSS sparer mye arbeid!

Stildefinisjonene lagres normalt i eksterne .css-filer.
Med en ekstern stilarkfil kan du endre utseendet til en hel nettside ved å endre bare én fil!
